# SSCReF-MAOF
SSCReF-MAOF serves as an intelligent decision-support tool for wastewater treatment plants, offering tailored designs for sulfur-based fillers.   It enables operators to navigate complex influent conditions and identify optimal formulations that satisfy strict discharge standards while minimizing operational costs.
Download and install the MATLAB software and the above-mentioned files. Double-click app.malpp to use SSCReF-MAOF.
